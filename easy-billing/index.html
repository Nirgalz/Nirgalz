Easy billing

Simply keep track of your bills and clients, print them in pdf.

The bill layout is made for one specific context, but can be customized using html and css.

This app is not maintained anymore and may contain security issues that can be resolved by updating dependencies. Also, it shouldn't be used as the main way to store history of professional bills without securing how data is saved.
